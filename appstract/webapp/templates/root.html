AppStract

Enter the url of a github repository to check out its AppStract !


It can be an overwhelming experience to join an open source project that has lots of code in it. You might start by reading documentation or even dive into the code directly. But sometimes you just want a quick overview. AppStract provides the solution.

AppStract analyzes go code from a github repository. This analysis results in a graph that visualizes the structure of the go program. In the graph two functions are connected if one function calls another, thus creating an abstract for an entire go program.

This application is created as part of the Gopher Gala Hackathon by Wieger Steggerda
