About Lirix


Lirix is a browser-based weather application that runs as a local webserver. It was created by Josh Vega (@Jsvcycling) for Gopher Gala 2015.

Lirix is written entirely in Go. Twitter Bootstrap is used for UI styling.

All data is obtained from OpenWeatherMap.
